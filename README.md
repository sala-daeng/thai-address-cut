# thai-address-cut

### ** npm i thai-address-cut **
```js
import thaiAddressCut from ‘thai-address-cut’ 
```
or 
```js
const thaiAddressCut = require(‘thai-address-cut’) 
```

how to use 
```js
const output = thaiAddressCut.cut({address(string)}, fullSearch(boolean : default = true))
```
